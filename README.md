# diskpulse-for-mac
DiskPulse for Mac helps you understand where disk space goes. Scan your Mac, explore folders in a clear Finder-like tree, sort them by size or growth, compare snapshots over time, and open any folder directly in Finder. Fast, private, local analysis. Помогает понять, куда уходит место на диске. Изучайте папки в наглядном дереве, сортируйте их.
