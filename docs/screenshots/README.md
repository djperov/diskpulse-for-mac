# Screenshots

Capture screenshots from the current release before publishing it:

1. Run DiskPulse for Mac and complete a scan.
2. Show the expanded folder tree and size/growth columns.
3. Save an English image as `overview-en.png` and a Russian image as `overview-ru.png` in this folder.
4. Add both images to the main README.

This avoids publishing screenshots of an outdated interface.
